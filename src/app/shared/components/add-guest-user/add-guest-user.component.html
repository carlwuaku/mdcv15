Two-factor authentication

Two-factor authentication is enabled by default. The instructions on how to set it up can be sent to each user from the user management section at any time.

You need to choose a deadline by which the users need to set up their two-factor authentication. After this date, the users will not be able to log in until they set up their two-factor authentication. The deadline can be changed at any time from the user management section.
